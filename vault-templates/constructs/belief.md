# Belief

> The mental models and assumptions you hold about the world, yourself, and others — and how they update.
> This note is a lens, not a test — a place to file what you notice, and the frame agents
> use to read your daily notes.

## What to notice (for your notes)
- A belief that drove an action this week (stated or implied).
- A belief that met counter-evidence — did it update or hold?
- Assumptions you're treating as facts.

## Related constructs
- `identity` — self-beliefs are a sticky subclass.
- `decision-making` — beliefs set the priors decisions run on.
- `emotion` — affect protects beliefs from updating.
- `metacognition` — noticing a belief is the first step to testing it.
