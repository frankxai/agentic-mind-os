# 00 — Start Here

Welcome to your Agentic Mind OS vault.

## The System

This vault is your second brain + self-intelligence OS.

Folders:
- 01_Daily — captures, journals, quick notes
- 02_Sources — books, papers, articles, podcasts (distilled)
- 03_Concepts — atomic concepts and definitions
- 04_Patterns — recurring patterns you notice
- 05_Learning — active recall, spaced reps, projects
- 06_Projects — active work
- 07_Identity — values, roles, future self, canon
- 08_Reviews — weekly, monthly, quarterly reflections

## How to Use with Agents

1. Capture in 01_Daily using the daily-mind-capture skill.
2. Distill sources using source-distillation skill.
3. Run weekly-review workflow.
4. Invoke agents like @mind-cartographer on your notes.
