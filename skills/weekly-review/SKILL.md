# weekly-review

## Trigger
End of week or user request for reflection.

## Inputs
- Notes from 01_Daily last 7 days
- Patterns identified
- Goals from 07_Identity

## Steps
1. Run pattern-analyst on the week.
2. Summarize wins, blocks, energy trends.
3. Update personal canon (07_Identity).
4. Generate 3-5 active recall items.
5. Plan next week (08_Reviews).

## Outputs
- Weekly review note in 08_Reviews/
- Updated canon entries
- New tasks in 06_Projects

## Invocation
"Run weekly-review on last 7 days of captures."
