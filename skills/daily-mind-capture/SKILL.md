# daily-mind-capture

## Trigger
User wants to capture thoughts, events, insights, or feelings from the day.

## Inputs
- Date or time period
- Raw stream (voice, text, bullet points)

## Steps
1. Listen or read the raw input.
2. Structure into:
   - Events / actions
   - Insights / thoughts
   - Emotions / energy
   - Open questions
3. Write atomic notes into 01_Daily/ with proper schema.
4. Tag with date and context.
5. Suggest 1-2 follow-ups (distill, review, canon update).

## Outputs
- Structured daily notes
- Updated daily log
- Suggested next skills

## Example
User: "Today I read about spaced repetition and realized I forget most things after 2 days."

Output: note in 01_Daily/ with tags #learning #memory , linked to 05_Learning.

## Invocation
"Run daily-mind-capture on this transcript: [paste]"
