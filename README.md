# Agentic Mind OS

**Agentic Mind OS** is a local-first second brain and self-intelligence operating system for students, creators, founders, and lifelong learners.

## What it is

A structured vault + agent + skill + workflow system that helps you:

- Capture daily mind activity without friction
- Distill sources into personal knowledge
- Build and maintain a personal canon
- Run regular reviews and reflection
- Use specialized agents for cartography, learning architecture, pattern analysis, and memory governance

## Core Components

- **Vault Template** — opinionated but flexible folder structure (see vault-template/)
- **Agents** — specialized markdown-defined agents (agents/)
- **Skills** — reusable, invocable capabilities (skills/)
- **Workflows** — end-to-end processes (workflows/)
- **Schemas** — data contracts for notes, patterns, reviews

## Installation (v0)

1. Clone this repo
2. Copy `vault-template/` into your local vault (Obsidian, Logseq, or plain markdown)
3. Use the agents and skills with your preferred coding agent (Codex, Claude, Hermes)
4. Start with `vault-template/00_Start_Here.md`
