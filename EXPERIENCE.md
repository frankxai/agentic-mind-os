# User Experience in Agentic Mind OS

**Positioning**: Your local-first second brain and self-intelligence operating system.

## Daily Experience (Lived OS)

Morning:
- Open 01_Daily and run daily-mind-capture skill (voice or text).
- Quick energy/mood log + 3 key events.
- Agents surface 1-2 insights from yesterday's canon.

Mid-day:
- Capture sources in 02_Sources.
- Run source-distillation skill → atomic notes in 03_Concepts or 04_Patterns.
- Learning-architect suggests 10-min active recall.

Evening:
- 08_Reviews quick reflection.
- mind-cartographer maps connections.
- Weekly review workflow on Fridays.

## Onboarding Journey (First 7 Days)

Day 1: Copy vault-template into your notes app. Read 00_Start_Here.md. First daily capture.
Day 2-3: Distill one source. Explore 07_Identity (start personal canon).
Day 4-7: Run first weekly-review. Invite agents via prompts.

## How It Feels

- **Calm structure**: Folders guide without rigidity.
- **Agent-augmented**: You capture; agents organize, connect, recall.
- **Personal canon grows**: Your "self" becomes queryable over time.
- **Local & private**: No cloud lock-in.

## Agent Exploration Paths (How Agents Navigate)

Agents start in 00_Start_Here or .codex/tasks.md.

Example prompts:
- "@mind-cartographer map connections in last 30 days of 03_Concepts"
- "@learning-architect build 14-day plan from my belief.md notes"
- "Run weekly-review on 01_Daily/ last week and update 07_Identity"

Exploration tree:
Vault Root → 01-08 folders → schemas for structure → agents/ and skills/ for automation.

Usefulness: Students retain 3x more via active recall + patterns. Founders track decision patterns. Lifelong learners build a living personal model of self.
